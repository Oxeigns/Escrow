Escrow Bot Ultimate - Codex Instructions
=======================================

1. Create a Telegram bot via @BotFather and obtain the token.
2. Copy `.env.example` to `.env` and fill in:
   - `BOT_TOKEN`: Telegram bot token
   - `OWNER_ID`: Telegram user id of the bot owner
   - `ADMIN_IDS`: Comma-separated Telegram user ids with admin access
   - `USE_WEBHOOK`: Set `true` to enable webhook mode or `false` for polling
   - `MONGO_URL` and `MONGO_DB_NAME`: MongoDB connection details
3. Run the bot locally:
   ```bash
   pip install -r requirements.txt
   python bot/main.py
   ```
4. For deployment to Render.com use the provided `render.yaml` and Dockerfile.
5. Use `admin/cli.py` for administrative tasks like listing users or resolving disputes.
